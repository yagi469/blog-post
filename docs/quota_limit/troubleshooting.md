# AntigravityQuotaWatcher トラブルシューティング

CLI上ではインストールが正常に完了し、拡張機能フォルダ(`~/.vscode/extensions/wusimpl.antigravity-quota-watcher-1.0.2`)も存在することを確認しました。
しかし、VS CodeのUI上に表示されない場合、以下の手順でトラブルシューティングをお願いします。

## 1. VS Code の再起動 (Reload Window)
最も一般的な解決策です。
1. `F1` または `Ctrl+Shift+P` を押してコマンドパレットを開きます。
2. `Developer: Reload Window` (または `開発者: ウィンドウの再読み込み`) と入力して実行します。
3. 再起動後、画面右下のステータスバーを確認してください。

## 2. 拡張機能の有効化確認
1. 左側のアクティビティバーから「拡張機能」アイコンをクリックします (または `Ctrl+Shift+X`)。
2. 検索バーに `Antigravity Quota Watcher` と入力します。
3. 拡張機能が表示されたら、「有効にする (Enable)」ボタンが表示されていないか確認してください。
   - 「無効 (Disabled)」になっている場合は有効化してください。
   - 「再読み込みが必要です (Reload Required)」と表示されている場合はボタンを押してください。

## 3. 手動でのポート再検出
まれに、Antigravity（IDEの言語サーバー）との通信ポートの検出に失敗することがあります。
1. コマンドパレット (`Ctrl+Shift+P`) を開きます。
2. `Antigravity Quota Watcher: Re-detect Port` を実行します。
3. 画面右下にスキャン中のメッセージが表示され、成功すれば配額が表示されます。

## 4. 設定の確認 (Windows ユーザー向け)
ポート検出がうまくいかない場合、設定の変更が有効な場合があります。
1. `Ctrl+,` で設定を開きます。
2. 検索バーに `antigravity-quota-watcher` と入力します。
3. `Force Power Shell` の設定項目がある場合、チェックを入れ替えてみて（オンならオフ、オフならオン）、再度「Re-detect Port」を試してください。

## 5. ログの確認
それでも解決しない場合、拡張機能のログを確認することで原因が特定できるかもしれません。
1. 表示 > 出力 (View > Output) を選択します。
2. 出力パネルのドロップダウンメニューから `Antigravity Quota Watcher` を探して選択します。
3. エラーメッセージが表示されていないか確認してください。
